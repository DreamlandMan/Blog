2. 设置本机绑定SSH公钥，实现免密码登录！（免密码登录，这一步挺重要的，码云是远程仓库，我们是平时工作在本地仓库！)

   ```bash
   # 进入 C:\Users\Administrator\.ssh 目录
   # 生成公钥
   ssh-keygen -t rsa -C '邮箱地址'
   ```

   ![](18.png)

3. 将公钥信息public key（C:\Users\xxx\\.ssh\id_rsa.pub）添加到码云账户中即可！
